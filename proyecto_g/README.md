# Automatización Trello - n8n - Telegram

## Descripción
Proyecto de automatización que envía notificaciones en tiempo real a Telegram cuando se crea una nueva tarjeta en Trello.

## Herramientas
- n8n
- Trello API
- Telegram Bot API

## Funcionamiento
Trello Trigger → n8n → HTTP Request → Telegram

## Evidencia
Se incluyen capturas del workflow y del mensaje recibido.

## Conclusión
El sistema permite monitorear tareas en tiempo real mediante automatización sin intervención manual.
